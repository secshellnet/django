PyTemplate supports the following databases:
  • MariaDB / MySQL
  • PostgreSQL
  • SQLite3 (for testing purpose only)
You can use the following authentication mechanics:
  • Django Authentication (using local database)
  • LDAP Authentication
  • OIDC (OpenID Connect: implemented with Discord, but I suggest something like Keycloak)
